**brand** <span style={{color:" #7d8793"}}>String</span>

The card issuer. 

<br />

**last4** <span style={{color:" #7d8793"}}>String</span>

Last 4 digits of card number.

Example value: 6345

<br />

**first6** <span style={{color:" #7d8793"}}>String</span>

First 6 digits of card number 

Example value: 411111

<br />

**cardType** <span style={{color:" #7d8793"}}>String</span>

The possible values are:

`DC` - Debit Card

`CC` - Credit Card

<br />

**issueCountry** <span style={{color:" #7d8793"}}>String</span>

The country where the card was issued.

Example value: AE

<br />

**cardLevel** <span style={{color:" #7d8793"}}>String</span>

The card tier or level, indicating the card's product category.

Example values: `PLATINUM`, `GOLD`, `CLASSIC`, `SIGNATURE`

<br />

**cardToken** <span style={{color:" #7d8793"}}>String</span>

A tokenized representation of the card that can be used for future transactions without re-entering card details.

<br />

**cardTokenExpTime** <span style={{color:" #7d8793"}}>Timestamp(3)</span>

The expiration time of the card token in milliseconds since epoch.

Example value: 1735689600000

<br />

**expMonth** <span style={{color:" #7d8793"}}>String</span>

Two-digit number, representing the card expiry month.

Example value: 01

<br />

**expYear** <span style={{color:" #7d8793"}}>String</span>

Two-digit number, representing last two digits of the card expiry year.

Example value: 22
