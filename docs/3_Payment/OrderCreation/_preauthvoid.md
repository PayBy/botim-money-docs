**PREAUTHVOID**

Cancel a pre-authorization and release the held funds before the transaction is captured.

---

**<font color="#333333">PayScene parameters</font>**

- **preauthOrderNo** <span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>

  The order number of the pre-authorization transaction to be cancelled.

  Example value: `PA202311190001234567`
